ローカル ドライブ ダッシュボード (Local Drive Dashboard)

このアプリケーションは、Googleドライブのファイルリンクを簡単に整理・管理できる、完全ローカルで動作するダッシュボードアプリです。

🌟 特徴

完全ローカル動作: サーバーや外部データベースは一切不要です。データはすべてお使いのブラウザの localStorage に安全に保存されます。

シングルファイル設計: HTML、CSS（Tailwind CSS）、JavaScriptが1つのHTMLファイルにまとまっているため、ファイルの移動や共有が非常に簡単です。

自動ファイルタイプ判定: 登録したGoogleドライブのURLから、ファイルの種類（スプレッドシート、ドキュメント、スライド、フォームなど）を自動的に推測し、適切なアイコンとテーマカラーを適用します。

カテゴリ分類と検索: ファイルを任意のカテゴリに分けて管理でき、キーワード検索で目的のファイルをすぐに見つけることができます。

レスポンシブデザイン: スマートフォンやタブレット、PCなど、どのデバイスでも見やすいレイアウトが自動で適用されます。

🚀 使い方

1. 準備

提供された drive-dashboard.html ファイルをパソコンの任意の場所に保存します。

保存したファイルをダブルクリックして、お使いのWebブラウザ（Chrome, Edge, Safariなど）で開きます。

2. ファイルの登録

画面右上の「新規登録」ボタン（スマートフォンの場合は画面下部など）をクリックします。

入力フォームが表示されます。

ファイル名: 管理しやすい任意の名前を入力します（例：月間売上レポート）。

Google ドライブ URL: Googleドライブで対象のファイルを開き、ブラウザのURLバーにあるアドレスをコピーして貼り付けます（または「共有」から取得したリンク）。

カテゴリ: 分類用のタグを入力します。入力候補から選ぶことも、新しく入力することも可能です（例：経理、企画、個人）。

「保存する」をクリックすると、ダッシュボードにファイルが追加されます。

3. ファイルを開く・管理する

開く: ダッシュボード上のカードをクリックすると、そのファイルが新しいタブで開きます。

検索・絞り込み: 画面上部の検索バーや、左側のカテゴリメニューを使って、ファイルを探すことができます。

編集・削除: 各カードにマウスカーソルを合わせる（スマートフォンの場合はタップする）と、右上に「編集」「削除」ボタンが表示されます。ここから情報の上書きやリストからの削除を行えます（※ここでの削除はダッシュボード上のリンクの削除であり、実際のGoogleドライブ上のファイルは削除されません）。

🛠️ 技術スタック

HTML5

CSS Framework: Tailwind CSS (CDN経由で読み込み)

JavaScript: Vanilla JS (フレームワーク非依存)

データ保存: Web Storage API (localStorage)

⚠️ 注意事項

このアプリはブラウザの localStorage を使用してデータを保存します。そのため、ブラウザのキャッシュや閲覧履歴を完全に削除すると、登録したデータも消去される可能性があります。

別のブラウザや別のパソコンで開いた場合、データは共有されません（それぞれのブラウザごとに保存されます）。

登録するのは「URL（リンク）」のみです。実際のファイルの内容をダウンロードしたり、このアプリ上で直接編集したりするものではありません。

オフライン状態では、Tailwind CSSのCDN読み込みができないため、デザインが崩れる場合があります。
